# Naming Conventions
## Types (class, struct)
Use Upper camel case (Pascal case)

Examples: `MyClass`

## Method
Use snake case

Examples: `save()`, `get_variable()`

## Variable
User snake case

Examples: `variable`, `my_variable`

## Constant and enum
User Upper camel case

Examples: `Constant`, `ConstVar`

## Namespace
User snake case

Examples: `my_name_space`
